# 文献追踪报刊风格 HTML 生成提示词

## 任务描述

帮我做一份《经济学人》/FT 财经社论报刊风格的「知网核心期刊文献追踪周报」单文件 HTML。

## 输入数据

- 数据来源：`文献追踪/文献追踪_YYYYMMDD.md`（由 cnki_rss_fetch.py 自动生成）
- 数据格式：Markdown，包含本期新增文献总数、覆盖期刊数、舞蹈/国标舞相关论文（重点关注区）、分刊详情（每篇含标题、作者、日期、摘要）

## 输出要求

### 视觉风格

- **底色**：米白浅暖底（#faf7f2 或 #f5f1eb）
- **字体**：衬线大标题（Noto Serif SC / STSong / SimSun），正文同色系
- **主色**：暗红（#8b1a1a）作为点缀色，用于重点标题、指标数字、首字下沉
- **排版**：多栏报刊排版（两栏或三栏 grid），首字下沉（first-letter 伪元素）
- **分隔**：用细线（border-top/bottom）和双线（4px double）做刊头、分节
- **无 emoji**：全部用文字和 SVG 图表表达

### 内容结构

#### 1. 刊名报头与导语摘要
- **报头**：
  - 英文刊名（如 Academic Frontier Observer）
  - 中文刊名（如 学术前沿观察）
  - 副标题（体育科学与舞蹈研究核心期刊文献追踪周报）
  - 期号、日期、数据来源、"内部参考"标识
- **导语**：
  - Kicker（小标题，如"本期概览"）
  - 大标题（概括本期核心发现，如"体育学术版图持续扩张，舞蹈研究走向跨学科深水区"）
  - 副标题（standfirst，1-2 句话总结本期亮点）

#### 2. 核心指标条
- 用 grid 布局（4 列），每个指标包含：
  - 大数字（暗红色，如 105）
  - 小标签（灰色，如"本期新增文献"）
- 指标示例：
  - 本期新增文献总数
  - 覆盖期刊数
  - 舞蹈相关论文数
  - 北京舞蹈学院学报篇数（或其他重点期刊）

#### 3. 正文（社论风格）
- **两段正文**，每段首字下沉、暗红色
- **主线一**：总量与结构分析（如"7 本期刊每刊稳定贡献 15 篇以上，AI 介入体育场景的研究数量显著上升"）
- **主线二**：舞蹈/国标舞研究动向（如"从'舞蹈中国性'的概念建构到 AI 编舞的美学讨论，呈现从经验总结向理论建构跃迁"）
- 语言风格：学术观察 + 社论评论，有观点、有判断，不是简单罗列

#### 4. 内联 SVG 图表（2-3 个）
- **图表一**：各期刊文献分布柱状图
  - X 轴：期刊名
  - Y 轴：篇数
  - 柱子颜色：暗红色（#8b1a1a），透明度渐变区分不同期刊
  - 柱子上方标注数字
- **图表二**（可选）：舞蹈相关论文占比饼图或环形图
- **图表三**（可选）：近几期文献数量趋势折线图（如果有历史数据）
- 每个图表下方标注"数据来源：知网公开 RSS 抓取，YYYY年MM月DD日"

#### 5. 重点关注区（舞蹈/国标舞相关）
- 用卡片样式（左侧暗红色竖线，浅灰背景）
- 标题："本期最值得精读的 3 篇"
- 列出 3 篇舞蹈相关论文，每篇包含：
  - 标题（可点击跳转知网链接）
  - 期刊名、作者、日期（灰色小字）

#### 6. 结构性议题卡片（3 个）
- 用 grid 布局（3 列），每个卡片包含：
  - 罗马数字编号（I、II、III）
  - 议题标签（如"AI 介入编舞"、"学科自主性"、"国标舞剧化"）
  - 1-2 句话描述（从本期论文中提炼的结构性议题）

#### 7. 分刊文献速览
- 每个期刊一个区块，包含：
  - 期刊名（大标题）
  - 篇数标签（如"15 篇"，暗红色背景）
  - 文献列表（编号、标题、作者、日期）
  - 每篇标题可点击跳转知网链接

#### 8. 页脚
- 数据来源与说明：
  - 数据来源：中国知网（CNKI）公开 RSS 订阅接口，抓取时间
  - 覆盖期刊列表
  - 去重机制说明
  - 完整数据另存为 Markdown 文件路径
  - 免责声明（摘要为原文前 120 字截断，完整内容请点击知网链接）
- 底部居中：英文刊名 + "EST. 2026 · FOR INTERNAL REFERENCE ONLY"

### 技术要求

- **单文件 HTML**：所有 CSS/JS 内联，无外链
- **响应式**：用 `@media (max-width: 720px)` 适配移动端，grid 布局在小屏下转为单列
- **无 emoji**：全部用文字和 SVG 图表表达
- **简体中文**：所有文本用简体中文
- **可点击链接**：所有论文标题用 `<a href="知网链接" target="_blank">` 包裹

### 适配文献追踪的特点

- **舞蹈相关论文自动置顶**：从 Markdown 的"重点关注"区提取，放入报刊的"重点关注区"和"结构性议题卡片"
- **分刊详情**：每个期刊一个区块，列出所有论文（或精选 3-5 篇）
- **摘要截断**：Markdown 中的摘要已经是 120 字截断，直接引用即可
- **知网链接**：每篇论文的 Markdown 链接直接转为 HTML `<a>` 标签

### 语言风格

- **学术观察 + 社论评论**：不是简单罗列数据，而是提炼趋势、给出判断
- **示例句式**：
  - "从本期抓取数据看，体育学术研究正沿着两条主线并行推进……"
  - "更值得关注的是舞蹈研究领域的新动向……"
  - "这对于舞蹈学科从'经验传承'走向'知识生产'的转型，具有指标性意义。"

## 使用示例

```
输入：文献追踪/文献追踪_20260713.md

输出：文献追踪/文献追踪_20260713.html（单文件，报刊风格）
```

运行 cnki_rss_fetch.py 后，把生成的 Markdown 文件路径给我，我用这个提示词生成 HTML。

---

## 关键变量

- **期号**：从日期计算，如 2026-07-13 是第 28 期（需根据实际起始日期调整）
- **抓取时间**：从 Markdown 文件的"生成时间"提取
- **舞蹈相关论文**：从 Markdown 的"重点关注"区提取
- **覆盖期刊**：从 Markdown 的分刊详情标题提取（如"体育科学（15 篇）"）

---

## 注意事项

1. **不要编造数据**：所有数字必须来自 Markdown 文件
2. **不要遗漏重点论文**：舞蹈/国标舞相关论文必须全部列出
3. **不要硬编码链接**：所有知网链接必须从 Markdown 提取
4. **保持报刊风格**：语言要像《经济学人》/FT 的社论，不是学术报告的干巴巴表述
