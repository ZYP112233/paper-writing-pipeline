# 文献综述审核清单

**理论基础**：文献综述三结构（综述背景+综述主体+综述评价）  
**审核对象**：人文社科领域学术论文文献综述  
**审核目标**：确保文献综述结构完整、分类科学、论证有力、评价深入

---

## 一、文献综述结构总览

```
┌─────────────────────────────────────────────────────────┐
│                    文献综述完整结构                       │
├─────────────────────────────────────────────────────────┤
│  第一部分：综述背景                                       │
│    ├── 理论型背景 / 历史型背景 / 概念型背景              │
│    └── 功能：建立综述的宏观视野                          │
├─────────────────────────────────────────────────────────┤
│  第二部分：综述主体（核心）                               │
│    ├── 文献分类（分论点）                                │
│    │   ├── 从上到下（选题出发）                          │
│    │   ├── 从下到上（文献出发）                          │
│    │   ├── 从老到新（时间脉络）                          │
│    │   ├── 从X到Y（变量关系）                            │
│    │   └── 由点到面（文献计量）                          │
│    └── 论证（叙述型/例证型）                             │
├─────────────────────────────────────────────────────────┤
│  第三部分：综述评价                                       │
│    └── 破界创新模型（5个问题）                           │
│        ├── 现有逻辑体系和基石假设                        │
│        ├── 现有本质认知                                  │
│        ├── 基石假设的真理性                              │
│        ├── 新假设/相反假设                               │
│        └── 新假设是什么                                  │
└─────────────────────────────────────────────────────────┘
```

## 二、第一部分：综述背景审核

### 2.1 背景类型判定

| 审核项 | 判定标准 | 状态 | 修改建议 |
|--------|----------|------|----------|
| **背景类型明确** | 属于哪种类型？ | □ | |

**三种背景类型对照表**：

| 类型 | 写作逻辑 | 适用场景 | 审核要点 |
|------|----------|----------|----------|
| **理论型背景** | A是重要话题→B/C/D多种观点（描述）→富有洞见（评价）→分三类（论题） | 理论争议较大领域 | 观点梳理+分类框架 |
| **历史型背景** | A是B阶段产物→C阶段变化（描述）→情况改变（评价）→D/E/F三种视角（论题） | 历史演变明显领域 | 阶段划分+演变逻辑 |
| **概念型背景** | A是关注概念→A概念是...→集中于B/C/D维度（描述）→E维度不够（评价）→三类视角（论题） | 概念界定关键领域 | 概念界定+维度分析 |

### 2.2 理论型背景审核

| 序号 | 审核项 | 审核标准 | 状态 | 修改建议 |
|------|--------|----------|------|----------|
| 1 | **话题重要性** | 是否说明A是重要研究话题？ | □ | |
| 2 | **观点梳理** | 是否梳理B、C、D等多种观点？ | □ | |
| 3 | **描述准确** | 对各观点的描述是否准确？ | □ | |
| 4 | **评价恰当** | 是否评价这些观点"富有洞见"？ | □ | |
| 5 | **分类框架** | 是否提出分类框架（如三类）？ | □ | |
| 6 | **逻辑完整** | 描述→评价→论题的逻辑是否完整？ | □ | |

**理论型背景模板**：
> [A]一直是[领域]研究的重要领域，许多学者从[角度]对[A]进行了深入观察和形象概括，提出了[B]、[C]、[D]等解释框架。（描述）这些研究大多[评价内容]。（评价）虽然观察到的现象不尽相同，但这些研究大多是从[角度1]和[角度2]两个角度出发对[A]进行解释。（论题）

**优秀案例**：
> 基层政府行为一直是我国政府组织研究的重要领域,许多学者从政策执行的全过程对基层政府的"拼凑应对"策略进行了深入观察和形象概括,提出了选择性执行、选择性应付、政策变通、上下共谋等解释框架。这些研究大多通过对单个案例的考察为我们描绘了基层政府政策执行波动的丰富图景,包括同一基层政府在执行不同政策时的行为选择,还包括其在同一政策不同执行阶段的行为变化。虽然观察到的现象不尽相同,但这些研究大多是从中国政府内部的激励机制和组织结构两个角度出发对基层政府行为进行解释。

### 2.5 综述背景常见问题

| 问题 | 表现 | 修改建议 |
|------|------|----------|
| **类型混淆** | 背景类型不清晰，混杂多种类型 | 明确选择一种类型，保持逻辑一致 |
| **描述缺失** | 缺少对文献的描述 | 补充对代表性文献的具体描述 |
| **评价空泛** | "富有洞见"等评价过于笼统 | 具体说明洞见所在 |
| **论题跳跃** | 从描述直接跳到论题，缺少过渡 | 补充评价环节，形成完整逻辑 |
| **分类模糊** | 提出的分类框架不清晰 | 明确分类标准和各类特征 |

**叙述型论证示例**：
> 关于第一方面，基于经验的调度方法能够快捷的对调度系统的问题进行判断，做出定性决策，具有快速简便、成本低、实时性高等优势，但存在过分依赖决策者的经验和智慧等缺陷，个体依赖性太强，知识显化过程困难，即使建立专家系统，其决策质量也受到知识库中知识数量以及这些知识质量的制约（Metaxiotis et al.，2002），在调度系统中的应用具有一定局限性（Framinan and Ruiz，2010）。

## 七、优秀文献综述案例

### 案例：理论型背景+从下到上分类+隐喻归类

**综述背景**：
> 基层政府行为一直是我国政府组织研究的重要领域,许多学者从政策执行的全过程对基层政府的"拼凑应对"策略进行了深入观察和形象概括,提出了选择性执行、选择性应付、政策变通、上下共谋等解释框架。这些研究大多通过对单个案例的考察为我们描绘了基层政府政策执行波动的丰富图景。虽然观察到的现象不尽相同,但这些研究大多是从中国政府内部的激励机制和组织结构两个角度出发对基层政府行为进行解释。

**综述主体**（从下到上+隐喻归类）：
> **3.1 引擎机制：基层政府行为的驱动力**
> （驱动因素组文献综述）
> 
> **3.2 路况调节：基层政府行为的约束条件**
> （调节因素组文献综述）
> 
> **3.3 驾驶技术：基层政府行为的策略选择**
> （策略因素组文献综述）

**综述评价**：
> 以往研究对基层政府行为做出了富有洞见的研究，但是在研究角度方面多从内部激励机制出发，相对忽视了外部制度环境的影响；在研究方法方面多采用单案例研究，缺乏多案例的比较分析。基于此，本研究将从制度环境的视角，采用多案例比较方法，对基层政府行为进行深入研究。
